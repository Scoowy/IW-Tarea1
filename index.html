HTML 5

¿5?

HTML5 es la última versión de HTML. El término representa dos conceptos diferentes:

  • Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos.
  • Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance.

Secciones y contenidos

Nuevas etiquetas que mejoran el esquema y secciones de un documento HTML5: <section>, <article>, <nav>, <header>, <footer>, <aside>.

Multimedia

Las nuevas etiquetas de <audio> y <video> permiten la manipulación de nuevos contenidos multimedia. Permite el uso, manipulación y almacenamiento de imagenes desde la cámara web. 

<video width="320" height="240" autoplay muted>
  <source src="video.mp4" type="video/mp4">
  <source src="video.ogg" type="video/ogg">
  Tu navegador no soporta la etiqueta video.
</video>

CSS 3

¿Qué hay de nuevo?

La especificación de CSS3 posee interesantes novedades que permitirán hacer webs más elaboradas y más dinámicas, con mayor separación entre estilos y contenidos. Dará soporte a muchas necesidades de las webs actuales, sin tener que recurrir a trucos de diseñadores o lenguajes de programación.

Características de diseño

Es posible combinar y apilar varios fondos, con la propiedad box-shadow se coloca una sombra al elemento. 

box-shadow: 0px 3px 7px 0px rgba(92, 92, 92, 0.45);

Mejora de bordes

Se pueden utilizar imágenes como estilos de los bordes con la propiedad border-image, mediante border-radius se puede redondear los bordes. 

border-left: 6px solid #d56f3e;
border-radius: 3px;

Sistema de animación de propiedades

Es posible constrollar las animaciones mediante el uso y modificación de las transiciones. 

transition-property: color;
transition-duration: 500ms;
transition-timing-function: ease-in-out;

Tipografía mejorada

Se tiene un mejor control en el uso de la tipografía, se puede modificar el texto, el color, tamaño, que sucede cuando el texto excede al contenido... 

font-family: "Montserrat", sans-serif;
color: #3D3D3D;
text-align: center;
font-size: 2.5em;
line-height: 2em;

Nuevas formas de maquetar

Nuevos sistemas de diseño, grid y flexbox, haciendo que estos sean "responive".

Estos sistemas de diseño permiten tener un mejor control sobre los elementos hijos, facilitando el trabajo del diseñador. 

display: flex;
flex-direction: row;
justify-content: center;
